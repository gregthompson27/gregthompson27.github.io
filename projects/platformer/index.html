Function Call Guidelines

  • createPlatform(x, y, width, height, color):
    Creates a platform at the specified position with the given dimensions and optional color.

  • createCollectable(type, x, y, gravity, bounce):
    Adds a collectable of the specified type at the given position with optional gravity and bounce values.

Examples:

  • createPlatform(500, 300, 200, 20, 'blue');
  • createCollectable('diamond', 200, 170, 0.5, 0.7);
